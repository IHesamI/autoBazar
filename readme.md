# Description
the car shopping project root 

## Clone the both projects
```bash

# clone both projects
$ git clone --recurse-submodules  https://github.com/IHesamI/autoBazar.git

# update the project

$ git submodule update --recursive --remote 

```



## Running the app

```bash
# install
$ npm run dep-install

# development in windows
$ npm run dev

# development in linux
$ npm run dev-linux

```
